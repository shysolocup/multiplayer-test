# Getting Started

## 1. Install Godot
specifically install the .NET version since this uses C#

https://godotengine.org/download/windows/

## 2. Install External Editor
Visual Studio Code is recommended but any editor supported by Godot should work

https://code.visualstudio.com/

## 4. Configure External Editor
follow the tutorial at https://docs.godotengine.org/en/stable/tutorials/editor/external_editor.html to configure VSCode for godot
