# Godot Coreblocks

A mini-engine built inside Godot comprising core building blocks for multiplayer P2P games innovating on the service-based structure of Roblox

Coreblocks improves upon Roblox's foundation by having Godot-specific systems

- service oriented design

- simple design

- built in scene-based map and lighting loading/saving

- multiple cameras with PhantomCamera

- accessible shaders (I'd recommend [https://godotshaders.com/](godotshaders.com) for resources)

- unified script nodes

- built-in discord rich presence

- improved input systems

- custom classes and services

- extensive multiplayer API integration 

- UDim2 integrations
